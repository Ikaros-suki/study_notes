# Swing 框架
* 抽象窗口工具包(Abstract Window Toolkit, **AWT**).

## 容器与组件
* Swing GUI 包含两种元素, <font color = darkred>组件和容器</font>

* **容器**是可以包含组件的特殊组件
  * 重量级容器 : 不能被其他容器包含
  * 轻量级容器 : 可以包含与被包含
