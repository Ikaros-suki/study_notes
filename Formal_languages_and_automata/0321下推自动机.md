# 下推自动机

* 等价于 上下文无关文法
* ε-NFA + 栈 = PDA
* 七元组 P = (Q, Σ, Γ, δ, q0, Z0, F):
   1. Q, 有穷状态集;
    1. Σ, 有穷输入字母表;
    2. Γ, 有穷栈字母表, 或栈符号集;
    3. δ : Q × (Σ ∪ {ε}) × Γ → $2^{QΓ∗}$, 状态转移函数;
    4. q0 ∈ Q, 初始状态;
  1. Z0 ∈ Γ − Σ, 初始符号, PDA 开始时, 栈中包含这个符号的一个实例, 用来表示栈底, 栈底符号之下无任何内容;
  2. F ⊆ Q, 接收状态集或终态集.
