# 马克思主义中国化的提出

* 马克思主义的诞生 : 1848年, **《共产党宣言》**
* 马克思主义中国化的**首次提出** : 1938年, 毛泽东, 六届六中全会, 《论新阶段》
* 马克思主义中国化, 是解决中国实际问题的客观需要

# 马克思主义中国化的理论成果

* 马克思主义深刻改变了中国, 中国也极大丰富了马克思主义
* 马克思主义中国化的两大成果 : **毛泽东思想** 与 **中国特色社会主义**
* 2018年, 十三届全国人大 马克思列宁主义, 毛泽东思想, 邓小平理论, "三个代表"重要思想, 科学发展观, 习近平新时代中国特色社会主义
